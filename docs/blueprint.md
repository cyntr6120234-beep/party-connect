# **App Name**: PartyConnect

## Core Features:

- Party Creation & Management: Organizers can create new party events with essential details like date, time, description, and location.
- Invitation Link Sharing: Generate a unique, shareable link for each party event, allowing guests to access details and RSVP without authentication.
- Guest RSVP Functionality: Guests can view party details via the invitation link and confirm their attendance (RSVP) directly on the page.
- Event Details Display: Provide a clear and concise view of all event information for both organizers and invited guests.
- Real-time Two-Way Chat: Enable real-time messaging between guests and organizers for live communication about the event.
- Guest Chat Name Entry: Allow guests to enter a temporary display name before joining the event's chat, since no authentication is used.
- AI Party Theme Tool: A generative AI tool to suggest unique and relevant party themes based on event type, date, or other details provided by the organizer.

## Style Guidelines:

- Primary color: A warm and inviting coral (#EE652B). The hue evokes a sense of vibrant social energy and warmth, perfect for celebrating connections.
- Background color: A subtle, warm off-white (#F4F1F0). It's a muted extension of the primary hue, providing a clean and approachable canvas.
- Accent color: A soft, complementary rosy hue (#E87D92). This analogous color adds a touch of playfulness and gentle emphasis to interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern readability and versatile, objective aesthetic suitable for both titles and longer text blocks like chat messages.
- Use clean, modern line-art icons that subtly hint at event themes, communication, and social interaction.
- Employ clean, modular card-based layouts for displaying event information, ensuring easy scanning and readability. The chat interface will feature a distinct, scrollable area for seamless conversation flow.
- Integrate smooth and subtle transition animations for interactive elements and state changes, enhancing user engagement without being distracting.
